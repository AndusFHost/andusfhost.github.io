How can i "create" a profile?

It's pretty easy.

1. Clone the repository

2. Copy the folder "templateuser" from "user" folder and change the name everywhere in index.html

3. Copy folder for your file ("templatefile") or files pack ("templatefiles") and add it there

4. Modify index.html in file folder like in step 2.

5. Push the commit to original repository
